# Lesson_4 Toán tử số học Arithmetic
    +       --> Cộng
    -       --> Trừ
    *       --> Nhân
    **      --> Lũy thừa
    /       --> Chia
    %       --> Chia lấy số dư
    ++       --> Tăng 1 giá trị số
    --       --> Giảm 1 giá trị số

# Lesson_5 Toán tử gán - Assignment
    Toán tử        Ví dụ              Tương đương
    =              x = y              x = y
    +=             x += y             x = x + y
    -=             x -= y             x = x - y
    *=             x *= y             x = x * y
    /=             x /= y             x = x / y
    **=            x **= y            x = x ** y
